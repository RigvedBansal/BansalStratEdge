Capital Allocation Tool

Capital Scoring & Prioritization Matrix

Capital deserves a common scoreboard. This matrix lets finance compare growth bets, efficiency projects, and operating fixes using one set of weights and one set of gating rules. That makes saying no less political and saying yes more defensible.

Best For

  • Quarterly capital planning and project ranking
  • Board discussions where multiple projects compete for cash
  • Resetting old projects through stop / continue / reset review

Core Outputs

  • Weighted score by project
  • Payback, risk, and execution-speed comparison
  • Fund now / phase / pilot / kill recommendation

Weight Table

Set one weight system for the quarter and do not change it project by project.

Criterion Weight What good looks like
Strategic fit 25% Directly supports core strategy, not side ambition
Expected ROI / payback 25% Clear economics with recoverable payback
Capital efficiency 20% Low capital intensity for the expected value created
Risk 15% Downside contained and understood
Execution speed 15% Can be implemented without major drag or dependence risk

Gating Rules

  • If payback is beyond the approved hurdle, require board-level justification.
  • If risk is high and execution readiness is low, force a pilot or defer outcome.
  • If score is low but project is already underway, run a formal stop / continue / reset review.
  • Always keep a visible kill list. It protects the ranking discipline.

Key Logic

Weighted score Weighted score = Sum of (criterion score x criterion weight)
Ranking rule Highest score wins only if gating rules are also passed
Kill-list filter If payback > hurdle and strategic fit low, default to do not fund

Common Mistakes

  • Changing weights to suit the preferred project
  • Using narrative to override weak economics every time
  • Skipping stop / continue / reset reviews for sunk-cost reasons

When This Tool Shines

Use it when several projects all sound important but capital is tight and finance needs a defensible ranking logic the business can still understand.
